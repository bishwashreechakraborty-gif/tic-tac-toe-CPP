# 🕹️ Professional Tic-Tac-Toe (C++)

A robust, console-based implementation of the classic Tic-Tac-Toe game. This project demonstrates core programming principles including **2D arrays**, **modular functional design**, and **dynamic UI rendering** in a terminal environment.

---

## 🚀 Features
* **Interactive Gameplay:** Two-player mode with input validation.
* **Dynamic Board:** The console refreshes after every move for a clean, "app-like" experience.
* **Win/Tie Detection:** Comprehensive logic to detect rows, columns, diagonals, and draw scenarios.
* **Replay System:** Allows players to restart the game without exiting the program.
* **Modular Code:** Written with high readability and maintenance in mind using separate functions for game logic.

## 🛠️ Technical Implementation
This project showcases several fundamental Computer Science concepts:
* **Data Structures:** Utilized `std::vector<vector<char>>` to represent a 3x3 game grid.
* **Algorithm Logic:** Custom logic for coordinate mapping (converting a 1-9 input into row/column indices).
* **Control Flow:** Implementation of `do-while` loops for game states and `for` loops for win-checking.
